Module Showcase

核心页面

解题工作台

单对话工作台支持上传图片 / PDF、结构化讲解、批改结果和后续追问，是学生使用频率最高的主界面。

  • 单对话模式，减少历史会话干扰
  • 支持表格、列表、讲解结构化输出
  • 附件上传与输入框合并在同一工作流里

学习计划

把今日状态、本周计划、明日建议、优先复习知识点与重点纠错方向整合成一页行动面板。

  • 学生能快速看到今天先学什么
  • 纠错方向和复习点明确分层
  • 界面强调行动建议而不是纯文档堆砌

知识图谱 / 错题图谱

知识图谱强调学习结构，错题图谱强调错误模式，两者共享同一套图形语言，但颜色和诊断语义不同。

  • 焦点视图突出当前主线
  • 节点标签和风险徽标有避让规则
  • 右侧详情卡承接掌握度、严重度和建议动作

运行状态 / 心跳 / Skills

运行状态页负责系统健康，心跳页负责自动任务节律，Skills 页负责附件回复中的附加输出风格。

  • 适合管理者快速判断系统是否稳定
  • 把可运行性、自动化和教学风格拆开管理
  • 页面样式保持统一，但语义明确区分
