<section class="section-block">
  <div class="section-header">
    <span class="section-kicker">工作流</span>
    <h2>从输入到回看，产品链路保持连续</h2>
    <p>用户感知到的不是一组离散工具，而是一条清楚的学习路径。每一步都为下一步提供上下文和结构化结果。</p>
  </div>
  <div class="timeline-grid">
    <article class="timeline-card">
      <span class="timeline-card__step">01</span>
      <h3>输入题目</h3>
      <p>学生通过图片、PDF 或 Web 工作台把学习任务送入系统，同一入口保留完整上下文。</p>
    </article>
    <article class="timeline-card">
      <span class="timeline-card__step">02</span>
      <h3>代理运行</h3>
      <p>MathClaw 负责诊断、讲解、批改和追问，在需要时追加更具体的附件输出风格。</p>
    </article>
    <article class="timeline-card">
      <span class="timeline-card__step">03</span>
      <h3>记忆沉淀</h3>
      <p>会话被整理成学习计划、知识图谱、错题图谱以及日记忆和周记忆，形成长期线索。</p>
    </article>
    <article class="timeline-card">
      <span class="timeline-card__step">04</span>
      <h3>管理回看</h3>
      <p>管理侧通过运行状态、心跳和控制页面判断系统是否稳定，确保学习链路持续运转。</p>
    </article>
  </div>
</section>
