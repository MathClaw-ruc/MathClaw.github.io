Quick Start

Quick start

The commands below follow the current repository layout and can be run directly from the source tree.

01

Install dependencies

python -m venv .venv
. .venv/bin/activate
pip install -e .
02

Prepare workspace

mkdir -p workspace/sessions
mkdir -p workspace/memory
mkdir -p workspace/cron
03

Start gateway

mathclaw gateway \
  --workspace ./workspace \
  --channel wecom
04

Start console

cd console
MATHCLAW_CONSOLE_WORKSPACE=../workspace \
python serve.py
