Local Installation ​

Install dependencies ​

bash
python -m venv .venv
. .venv/bin/activate
pip install -e .

Prepare the workspace ​

bash
mkdir -p workspace/sessions
mkdir -p workspace/memory
mkdir -p workspace/cron

Start the gateway ​

bash
mathclaw gateway \
  --workspace ./workspace \
  --channel wecom

Start the console ​

bash
cd console
MATHCLAW_CONSOLE_WORKSPACE=../workspace python serve.py

Default console address:

text
http://127.0.0.1:6006
